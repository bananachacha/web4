SEO 101 Basic

Written by. Yoonna cha

[On page SEO]

What is on page SEO?
웹 사이트의 검색 엔진 순위를 높이고 유기적 트래픽을 얻기 위해 웹 페이지를 최적화

Why is it essential?
검색 엔진이 웹 사이트와 콘텐츠가 검색 자의 검색어와 관련이 있는지 판단할 수 있다

How to do?
고품질 콘텐츠 게시, HTML 태그 수정, 이미지 최적화

{head}
1.title :
  • 해당 페이지의 주제 주제가 무엇인지 알려주는 초기 단서 역할
    (검색엔진 결과 페이지, 브라우저탭에 표시)
  • 핵심 키워드 반드시 포함하기. 가능하면 맨 앞에
  • 페이지 별 중요 키워드로 (60바이트 이내로)
  • unique, descriptive
2.meta :
  • 노출 순위와는 상관없지만, 클릭 유도에 효과적
  • meta name=”description” content=” ”
  • meta name=”keywords” content=” ”

{body}
1.h1 ~ h6 : 키워드 반복 금지
2.p
3.a
4.alt should be specific(125 or less)

[On server SEO]

1.URL :
  • 최대한 짧게
  • 특수기호 사용 금지
  • 페이지 주제어 반드시 포함하기
2.site speed :
이미지 용량, 자바스크립트 등 요인 분석해서 속도 개선

3.internal link easy to navigate and understand

4.반응형 웹 어떤 기기로 접속해도 호환이 잘 되도록

5.웹 표준 규약 브라우저 호환성

6.XML sitemap

7.robot.txt

[off page SEO]

Link-building
1.write guest blogs to promote yourself

2.make relevant, interesting, engaging, informative, educational and remarkable contents

3.write content that mentions influencers in your field

4.scour blogs with broken link and suggest replacing it with your content

5.take advantage of the popularity of infographics

[SEO 성과 측정 기준]

  • 노출 키워드 수 증가
  • 트래픽 증가
  • 페이지 등록 순위 개선
